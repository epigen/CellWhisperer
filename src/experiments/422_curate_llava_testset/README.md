We extracted 100 random samples/cells from each of the two datasets (cxg and archs4). for archs4, these represented diverse conversation types (detailed, complex, normal, phase1-dataset). These samples were (of course) excluded from training. We manually examined these 200 samples and curated them in few cases, where the transcriptomic state was not subject to the conversation.

The edits are tracked in the docx.
