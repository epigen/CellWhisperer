This folder provides an exploratory space for all our analyses.

If you start a new task, create a folder in here, in which you place your notebooks and files in a disorganized manner.

When finalizing your task, integrate the working code into the rest of the code base to make it usable for the team.

Make sure not to commit large files (e.g. >5MB in total) and use the PR system to get feedback (either when the task is final, or in the middle, if you can make use of feedback).
